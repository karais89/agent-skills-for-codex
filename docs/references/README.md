# 참고 문서 분류 기준

이 디렉터리는 외부 자료와 비교 분석 문서를 채택 상태에 따라 나누어 보관한다. 폴더 이름은 현재 하네스에 반영된 정도를 나타내며, 자료의 품질 평가가 아니라 이 프로젝트의 초기 범위에 대한 판단이다.

외부 클론도 같은 기준으로 `reference-repos/adopted/`와 `reference-repos/supporting/`에 둔다.

## 폴더

- `adopted/`: 현재 원칙과 작업 흐름의 기본축으로 채택한 참고 자료
- `supporting/`: 기본축은 아니지만 부분 채택, 보류, 향후 검토에 쓰는 보조 참고 자료
- `analyses/`: 채택 판단, 참고 저장소 목록, 비교 분석처럼 결정을 설명하는 문서

새 참고 자료를 추가할 때는 먼저 `analyses/`에 판단 근거를 남기고, 기본축으로 쓰기로 결정한 뒤에만 `adopted/`로 옮긴다.
