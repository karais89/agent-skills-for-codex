# 하네스 템플릿 릴리즈 기록

이 디렉터리는 full 하네스 템플릿의 릴리즈 후보와 릴리즈 기록을 남긴다. 대상 프로젝트에 복제되는 템플릿 문서가 아니라, 이 저장소에서 템플릿 상태를 고정하고 판단하기 위한 내부 운영 문서다.

## 기록 규칙

- 릴리즈 기록은 `harness-template-v<major>.<minor>.md` 형식으로 작성한다.
- 릴리즈 후보는 실제 태그를 만들기 전까지 상태를 `릴리즈 후보`로 둔다.
- 실제 태그를 만들면 같은 문서에 태그명과 태그 대상 커밋을 기록한다.
- 검증 명령, fixture 결과, 알려진 제약, 기존 프로젝트 적용 정책을 함께 남긴다.
